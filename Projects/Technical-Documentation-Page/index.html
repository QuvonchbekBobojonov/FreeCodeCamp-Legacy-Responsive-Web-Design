Introduction

JavaScript is a versatile programming language that runs in the browser and on servers using environments like Node.js.

It allows you to create interactive web pages, handle events, and communicate with servers using APIs such as fetch().

JavaScript is one of the core technologies of the web, along with HTML and CSS.

  • Client-side scripting language
  • Supports object-oriented programming
  • Used in both frontend and backend
Getting Started

The easiest way to start with JavaScript is to include it directly in an HTML file using the <script> tag.

You can also run JavaScript from the browser console, which is useful for quick tests and debugging.

A simple example of JavaScript in HTML looks like this: 

<script>
  console.log("Hello, JavaScript!");
</script>
  • Use a modern browser
  • Open DevTools to access the console
  • Experiment with simple commands
Syntax and Types

JavaScript supports several data types such as string, number, boolean, null, undefined, symbol, and object.

You can declare variables using let, const, or the older var keyword.

Example of variable declarations: 

let age = 18;
const name = "Alex";
var isAdmin = false;

Arrays and objects are very common structures in JavaScript programs.

  • let is block-scoped
  • const is used for constants
  • var is function-scoped
Control Flow

Control flow statements allow you to make decisions and repeat actions in your code.

Common statements include if, else, switch, for, and while.

Example using an if statement: 

let score = 90;
if (score >= 60) {
  console.log("You passed!");
} else {
  console.log("Try again.");
}

Loops are used to execute a block of code multiple times, such as iterating over an array.

  • if/else for branching logic
  • for loops for counting iterations
  • while loops for repeated checks
Functions

Functions in JavaScript are reusable blocks of code that perform a specific task.

They can be declared using the function keyword or as arrow functions using =>.

Example of a regular function: 

function add(a, b) {
  return a + b;
}

Example of an arrow function: 

const multiply = (a, b) => a * b;

Functions can take parameters and return values, making them powerful tools for structuring your code.

  • Use functions to avoid repeating code
  • Arrow functions are shorter to write
  • Functions can be passed as arguments
